# Consultar Mapas Auxiliares

Existem dois métodos para consultar os mapas auxiliares submetidos.

---

## 1. Consulta por Período Específico

# (GET: /api/reportes/mapa-auxiliar/:ano/:trimestre)

**Descrição:**  
Este serviço permite a consulta detalhada de todos os dados auxiliares submetidos (participantes, activos financeiros e imobiliários) para um determinado período.

**Parâmetros de URL:**
- `ano`: Ano do reporte.
- `trimestre`: Trimestre do reporte (1 a 4).

---

## 2. Consulta Geral com Filtros

# (GET: /api/reportes/mapa-auxiliar)

**Descrição:**  
Permite listar todos os mapas auxiliares submetidos pela entidade, com suporte a filtros opcionais.

**Parâmetros de Query (Opcionais):**
- `periodo`: Filtra por uma data de referência específica (YYYY-MM-DD).
- `inicio` e `fim`: Filtra por um intervalo de datas de referência.

---

## Regras de negócio relevantes

- **Perímetro de Acesso:** O utilizador só consegue consultar os dados da própria entidade (baseado no token de autenticação).
- **Dados Retornados:** A resposta inclui arrays agregados de `participantes`, `activosFinanceiros` e `activosImobiliarios`.

---

## Propriedades do Serviço

| Nome            | Valor                                                    | Descrição                           |
| --------------- | --------------------------------------------------------| ----------------------------------- |
| Método HTTP     | GET                                                     | Consulta de dados                   |
| Autorização     | Bearer Token                                            | Obrigatório                         |
| URL Produção    | https://reportes.cmc.ao/api/reportes/mapa-auxiliar/{ano}/{trimestre} | Endpoint de consulta |
| URL Homologação | https://reporteshml.cmc.ao/api/reportes/mapa-auxiliar/{ano}/{trimestre} | Endpoint de teste |
