# (POST: /api/reportes/mapa-auxiliar)

**Descrição:**  
Este serviço permite a submissão dos mapas auxiliares obrigatórios, nomeadamente o Mapa de Participantes, Mapa de Activos Financeiros e Mapa de Activos Imobiliários, conforme as normas de reporte vigentes.

A submissão inicial deve ser feita de forma agregada, contendo obrigatoriamente todos os registos para o período (ano e trimestre) especificado.

> **Observação:** Todo o reporte deve ser enviado em formato `.JSON`. O modelo Excel disponibilizado serve apenas como referência para estruturação de dados.

**URLs de referência de modelo (disponíveis em /exemplos):**

- Modelo Excel: [/exemplos/mapa-auxiliar-modelo.xlsx](/exemplos/mapa-auxiliar-modelo.xlsx)
- Modelo JSON (teste): [/exemplos/mapa-auxiliar.json](/exemplos/mapa-auxiliar.json)

---

## Contexto de utilização

- Destinado a entidades supervisionadas (ex: Sociedades Gestoras de OIC) para o reporte periódico de dados auxiliares à CMC.
- Deve ser utilizado sempre que houver novos dados relativos a participantes, activos financeiros ou activos imobiliários para o trimestre em curso.

## Regras de negócio relevantes

- **Unicidade:** Só é permitida uma submissão inicial por trimestre. Caso necessite de corrigir dados, deverá utilizar o serviço de actualização.
- **Formato:** Exclusivamente formato `.JSON`.
- **Campos Obrigatórios:** Na submissão inicial, todos os arrays (`participantes`, `activosFinanceiros`, `activosImobiliarios`) devem ser enviados e conter pelo menos um registo.
- **Penalidade por Mora:** Se o reporte for submetido após o encerramento do trimestre (conforme regra de `LOCK_DAY`), o sistema registará o envio com penalidade, e o status de resposta será `SUBMETIDO COM PENALIDADE`.
- **Autenticação:** Requer Bearer Token obtido no login.

## Campos de Entrada (Estrutura JSON)

### Objecto Principal
| Nome                | Descrição                          | Tipo  | Preenchimento |
| ------------------- | ---------------------------------- | ----- | ------------- |
| ano                 | Ano do reporte (ex: 2025)          | Int   | Obrigatório   |
| trimestre           | Trimestre (1 a 4)                  | Int   | Obrigatório   |
| participantes       | Lista de participantes do fundo    | Array | Obrigatório   |
| activosFinanceiros  | Lista de activos financeiros       | Array | Obrigatório   |
| activosImobiliarios | Lista de activos imobiliários      | Array | Obrigatório   |

### Participante (Item de `participantes`)
| Nome                     | Descrição                                 | Tipo    | Exemplo           |
| ------------------------ | ----------------------------------------- | ------- | ----------------- |
| nif_fundo                | NIF do fundo regulamentado                | String  | 000000000         |
| nome_fundo               | Nome do fundo regulamentado               | String  | ZPX               |
| categoria_up             | Categoria da UP                           | String  | Categoria A       |
| nome                     | Nome completo do participante             | String  | João Silva        |
| nif                      | NIF do participante                       | String  | 5001234567        |
| dt_subscricao            | Data da subscrição                        | Date    | 2025-09-15        |
| valor_nominal_up         | Valor nominal da UP                       | Decimal | 1000.00           |
| quantidade_up_subscritas | Quantidade de UPs subscritas              | Int     | 50                |
| valor_subscrito          | Valor total subscrito                     | Decimal | 50000.00          |
| valor_liquidado          | Valor já liquidado                        | Decimal | 50000.00          |
| valor_actual_up          | Valor actual da UP                        | Decimal | 52000.00          |
| nivel_risco              | Nível de risco                            | String  | Médio             |

### Activos Financeiros (Item de `activosFinanceiros`)
| Nome             | Descrição                                      | Tipo    | Exemplo              |
| ---------------- | ---------------------------------------------- | ------- | -------------------- |
| nif_fundo        | NIF do fundo regulamentado                     | String  | 000000000            |
| nome_fundo       | Nome do fundo regulamentado                    | String  | ZPX                  |
| conta            | Código da conta do activo                      | String  | 3210.10.001          |
| descricao        | Descrição do activo                            | String  | Obrigações do Estado |
| moeda            | Moeda de referência                            | String  | AKZ                  |
| dt_cambio        | Data do câmbio (se aplicável)                  | Date    | 2026-03-31           |
| cambio           | Taxa de câmbio utilizada                       | Decimal | 1.0                  |
| quantidade       | Quantidade ou montante                         | Decimal | 200                  |
| valor_de_mercado | Valor de mercado actual                        | Decimal | 240000.00            |
| valias           | Mais ou menos-valias                           | Decimal | 1000.00              |
| valor            | Valor base                                     | Decimal | 240000.00            |
| juro             | Juros acumulados                               | Decimal | 5000.00              |
| total            | Valor total (Valor + Juro)                     | Decimal | 245000.00            |
| percentagem      | Peso percentual na carteira                    | Decimal | 25.0                 |

### Activos Imobiliários (Item de `activosImobiliarios`)
