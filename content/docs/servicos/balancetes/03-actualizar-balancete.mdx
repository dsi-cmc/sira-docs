# (POST: /api/reportes/balancete/updateBalancete)

**Descrição:**  
Permite que a entidade supervisionada actualize dados de um balancete previamente submetido.

O token obtido no login deve ser enviado no cabeçalho `Authorization: Bearer <token>`.

---

## Contexto de utilização

- Destinado a correcção ou actualização de balancetes já submetidos.
- Pode enviar apenas grupos específicos (ex.: passivos) sem reenviar o balancete completo.

## Regras de negócio relevantes

- **Identificador de Movimento:** A actualização baseia-se no `num_conta_movimento`.
- **Flexibilidade:** Pode enviar apenas as categorias que necessitam de correcção.
- **Prazos:** Se a actualização for feita após o prazo regulamentar, será marcada como "fora do prazo".

---

## Propriedades do Serviço

| Nome            | Valor                                                             | Descrição                |
| --------------- | ----------------------------------------------------------------- | ------------------------ |
| Método HTTP     | POST                                                              | Actualização de dados    |
| Cabeçalho       | Content-Type: application/json                                    | Corpo em JSON            |
| Autorização     | Bearer Token                                                      | Tipo de autenticação     |
| URL Produção    | https://reportes.cmc.ao/api/reportes/balancete/updateBalancete    | Endpoint de actualização |
| URL Homologação | https://reporteshml.cmc.ao/api/reportes/balancete/updateBalancete | Endpoint de teste        |

---

## Campos de Entrada (Estrutura do Item)

| Nome                | Descrição                       | Tipo    | Preenchimento | Exemplo          |
| ------------------- | ------------------------------- | ------- | ------------- | ---------------- |
| num_conta           | Número da Conta Principal       | String  | Obrigatório   | 1.1              |
| conta_principal     | Descrição da Conta Principal    | String  | Obrigatório   | DISPONIBILIDADES |
| num_subconta        | Número da Subconta              | String  | Obrigatório   | 1.10.10          |
| subconta            | Descrição da Subconta           | String  | Obrigatório   | NUMERÁRIO        |
| num_conta_movimento | Número da Subconta de Movimento | String  | Obrigatório   | 1.10.10.01       |
| nome_conta          | Nome da Subconta de Movimento   | String  | Obrigatório   | Caixa Geral      |
| debito              | Valor de débito                 | Decimal | Obrigatório   | 60000.00         |
| credito             | Valor de crédito                | Decimal | Obrigatório   | 10000.00         |
| saldo               | Saldo resultante                | Decimal | Obrigatório   | 50000.00         |

---
