# (POST: /api/reportes/balancete)

**Descrição:**  
Permite que a entidade supervisionada envie o seu balancete financeiro para o sistema de reporte da CMC.

Para efectuar a submissão, o utilizador deve autenticar-se através do endpoint de login para obter um token de acesso. Esse token deve ser enviado no cabeçalho `Authorization: Bearer <token>`.

> **Observação:** Todo o reporte deve ser enviado em formato `.JSON`. O modelo Excel disponibilizado serve apenas como referência para estruturação de dados.

**URLs de referência de modelo:**

- Modelo Excel: [/exemplos/balancete-modelo-excel.xlsx](/exemplos/balancete-modelo-excel.xlsx)
- Modelo JSON (teste): [/exemplos/balancete-modelo-json.json](/exemplos/balancete-modelo-json.json)

---

## Contexto de utilização

- Destinado a entidades supervisionadas que reportam trimestralmente os balancetes financeiros à CMC.
- Deve ser usado sempre que houver um balancete completo para submissão, garantindo conformidade regulatória.

---

## Regras de negócio relevantes

- **Validação:** O sistema realiza uma validação de estrutura e saldos antes de aceitar o reporte.
- **Prazos:** Submissões trimestrais devem ser feitas até o dia 8 após o fim do trimestre. Caso contrário, serão registadas como "fora do prazo".
- **Unicidade:** Só é permitido um balancete por trimestre para cada fundo (ou para a própria entidade).
- **Grupos Obrigatórios:** Devem constar todos os grupos contabilísticos:  
  `activos, passivos, compensacaoActiva, compensacaoPassiva, fundosProprios, resultados, fluxosDeCaixa.`

---

## Propriedades do Serviço

| Nome            | Valor                                             | Descrição             |
| --------------- | ------------------------------------------------- | --------------------- |
| Método HTTP     | POST                                              | Submissão de dados    |
| Cabeçalho       | Content-Type: application/json                    | Corpo em JSON         |
| Autorização     | Bearer Token                                      | Tipo de autenticação  |
| URL Produção    | https://reportes.cmc.ao/api/reportes/balancete    | Endpoint de submissão |
| URL Homologação | https://reporteshml.cmc.ao/api/reportes/balancete | Endpoint de teste     |

---

## Campos de Entrada

### Parâmetros Iniciais

| Nome       | Descrição                                            | Tipo    | Preenchimento | Exemplo   |
| ---------- | ---------------------------------------------------- | ------- | ------------- | --------- |
| ano        | Ano do período de reporte (>= 2000)                  | Inteiro | Obrigatório   | 2025      |
| trimestre  | Trimestre do período de reporte (1 a 4)              | Inteiro | Obrigatório   | 3         |
| total_up   | Quantidade total de Unidades de Participação         | Decimal | Obrigatório   | 660170.00 |
| nome_fundo | Nome do Fundo (se aplicável)                         | String  | Opcional      | Fundo XYZ |
| nif_fundo  | NIF do Fundo (se aplicável)                          | String  | Opcional      | 540000000 |

### Estrutura do Item (para cada grupo contabilístico)

| Nome                | Descrição                       | Tipo    | Preenchimento | Exemplo          |
| ------------------- | ------------------------------- | ------- | ------------- | ---------------- |
| num_conta           | Número da Conta Principal       | String  | Obrigatório   | 1.1              |
| conta_principal     | Descrição da Conta Principal    | String  | Obrigatório   | DISPONIBILIDADES |
| num_subconta        | Número da Subconta              | String  | Obrigatório   | 1.10.10          |
| subconta            | Descrição da Subconta           | String  | Obrigatório   | NUMERÁRIO        |
| num_conta_movimento | Número da Subconta de Movimento | String  | Obrigatório   | 1.10.10          |
| nome_conta          | Nome da Subconta de Movimento   | String  | Obrigatório   | Numerário        |
| debito              | Valor de débito                 | Decimal | Obrigatório   | 50000.00         |
| credito             | Valor de crédito                | Decimal | Obrigatório   | 10000.00         |
| saldo               | Saldo resultante                | Decimal | Obrigatório   | 40000.00         |
