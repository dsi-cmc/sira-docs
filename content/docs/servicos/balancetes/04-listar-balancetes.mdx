title: Listar Balancetes por Trimestre
---

# (GET /api/reportes/balancete/:ano/:trimestre/:nif_fundo?)

**Descrição:**  
Permite que a entidade supervisionada obtenha a lista de balancetes submetidos, filtrando por ano e trimestre. O parâmetro `nif_fundo` é opcional e permite filtrar por um fundo específico.

O token do login deve ser enviado no cabeçalho `Authorization: Bearer <token>`.

---

## Contexto de utilização

- Consultar balancetes submetidos por período.
- Visualizar status e gerar relatórios internos.

---

## Propriedades do Serviço

| Nome            | Valor                                                                     | Descrição              |
| --------------- | ------------------------------------------------------------------------- | ---------------------- |
| Método HTTP     | GET                                                                       | Solicitação de dados   |
| Autorização     | Bearer Token                                                              | Tipo de autenticação   |
| URL Produção    | https://reportes.cmc.ao/api/reportes/balancete/{ano}/{trimestre}      | Listagem por trimestre |
| URL Homologação | https://reporteshml.cmc.ao/api/reportes/balancete/{ano}/{trimestre}   | Endpoint de teste      |

---

## Parâmetros de URL

| Nome      | Descrição                           | Tipo   | Preenchimento | Exemplo   |
| --------- | ----------------------------------- | ------ | ------------- | --------- |
| ano       | Ano do reporte                      | Number | Obrigatório   | 2025      |
| trimestre | Trimestre do reporte (1 a 4)        | Number | Obrigatório   | 2         |
| nif_fundo | NIF do Fundo (se aplicável)         | String | Opcional      | 540000000 |
