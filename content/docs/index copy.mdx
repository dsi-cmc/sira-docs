# Bem-vindo à Documentação da API de Reportes de Balancetes da CMC

Esta documentação destina-se a entidades supervisionadas que precisam enviar, consultar e actualizar informações financeiras através da API oficial da **Comissão do Mercado de Capitais (CMC)**.

O sistema permite a troca estruturada de dados em formato **JSON**, garantindo segurança, integridade e consistência das informações. Todos os processos de envio e consulta devem respeitar as regras de negócio estabelecidas.

---

## Navegação Rápida

### Serviços Principais

- [Login](./docs/reportes/balancetes/servicos/01-login) - Autenticação e obtenção de token de acesso.
- [Submeter Balancete](./docs/servicos/balancetes/02-submeter-balancete) - Submissão de balancetes financeiros.
- [Actualizar Balancete](./docs/servicos/balancetes/03-actualizar-balancete) - Correção ou atualização de balancetes existentes.
- [Listar Balancetes por Trimestre](./docs/servicos/balancetes/04-listar-balancetes) - Consulta de balancetes por ano e trimestre.

## Boas Práticas

- Sempre teste suas requisições em ambiente de homologação antes de enviar dados em produção.
- Utilize apenas o formato JSON para envio de balancetes.
- Mantenha seu token de autenticação seguro e actualizado.
- Valide todos os valores numéricos e campos obrigatórios de acordo com o plano contabilístico definido.
